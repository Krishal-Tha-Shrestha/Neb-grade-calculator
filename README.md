# 🎓 NEB Grade 12 GPA Calculator

A free, offline-capable GPA calculator for **Nepal Education Board (NEB) Grade 12 Science stream** students. Supports both **Biology** and **Computer Science** optional subjects and follows the official NEB grading system.

🌐 **Live Site:** [krishal-tha-shrestha.github.io/Neb-grade-calculator](https://krishal-tha-shrestha.github.io/Neb-grade-calculator/)

---

## ✨ Features

- 🧬 Switch between **Biology** and **Computer Science** streams
- ⚡ **Live grade preview** as you type marks
- 📊 Calculates **final GPA** and **Division** (Distinction / First / Second / Third / Fail)
- 📋 **Copy Result** button to copy your GPA to clipboard
- 🔗 **Share Result** button using Web Share API (mobile)
- 📱 Fully **responsive** — works on mobile and desktop
- ♿ **Accessible** — aria-labels, semantic HTML
- 🔍 SEO optimized

## 📐 NEB Grading System

| Percentage | Grade | GPA | Remark |
|---|---|---|---|
| 90 – 100 | A+ | 4.0 | Outstanding |
| 80 – 89  | A  | 3.6 | Excellent |
| 70 – 79  | B+ | 3.2 | Very Good |
| 60 – 69  | B  | 2.8 | Good |
| 50 – 59  | C+ | 2.4 | Satisfactory |
| 40 – 49  | C  | 2.0 | Acceptable |
| 35 – 39  | D  | 1.6 | Basic |
| Below 35 | NG | 0.0 | Non-Graded |

## 🚀 How to Use

**Online (recommended):**
Just visit the live site — no installation needed.

**Locally:**
1. Clone the repo:
   ```bash
   git clone https://github.com/Krishal-Tha-Shrestha/Neb-grade-calculator.git
   ```
2. Open `index.html` in any browser

That's it — no dependencies, no build steps, no internet required after loading.
